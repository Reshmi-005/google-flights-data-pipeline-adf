# Live Google Flights Search Pipeline using Azure Data Factory

## Project Overview

This project demonstrates an end-to-end ETL pipeline using Azure Data Factory.

### Workflow

SearchAPI.io (Google Flights API)
        ↓
Azure Data Factory
        ↓
Azure Data Lake Storage Gen2
        ↓
Mapping Data Flow
        ↓
Azure SQL Database
        ↓
Monitoring & Scheduling

## Technologies Used

- Azure Data Factory
- Azure Data Lake Storage Gen2
- Azure SQL Database
- SearchAPI.io
- REST API
- SQL

## Features

- REST API Integration
- JSON Data Ingestion
- Flatten Transformation
- Derived Columns
- Azure SQL Loading
- Error Handling
- Schedule Trigger
- Pipeline Monitoring
